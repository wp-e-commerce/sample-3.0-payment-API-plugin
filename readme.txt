A sample payment gateway.

== Description ==

This is a sample gateway for the 3.0 Payment Gateway API in WP eCommerce.

== Installation ==

1. Upload the folder 'sample-payment-gateway' to the '/wp-content/plugins/' directory
2. Activate the plugin through the 'Plugins' menu in WordPress
